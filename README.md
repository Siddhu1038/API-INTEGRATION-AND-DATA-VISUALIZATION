This project demonstrates how to fetch real-time data from a public API and visualize it using Python. The application uses the OpenWeatherMap API to retrieve weather information such as temperature and humidity for a selected city. The data is processed and displayed using graphical plots created with Matplotlib.
Additionally, an interactive dashboard is developed using Streamlit, allowing users to input a city name and view dynamic weather trends directly in a web browser. This project highlights practical skills in API handling, data processing, and data visualization, making it useful for real-world applications like weather monitoring systems.
